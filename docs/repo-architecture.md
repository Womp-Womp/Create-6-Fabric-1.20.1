# Repository Architecture

The `com.simibubi.create` package is organized into the following top-level packages:

- `api`: Public-facing interfaces and utilities for addon developers.
- `compat`: Integration and compatibility layers for other mods and tools.
- `content`: Gameplay content such as blocks, items, machines, and mechanical systems.
- `foundation`: Core framework, utilities, and shared systems underpinning the mod.
- `impl`: Internal implementation details supporting the public API.
- `infrastructure`: Configuration, commands, data generation, world generation, and other development infrastructure.
